# Publicar AMARTR3S

Este proyecto ya esta listo para publicarse como sitio estatico.

## Archivos principales

- `index.html`
- `catalogo.html`
- `assets/`
- `images/`

## Opcion 1: GitHub Pages desde navegador

1. Crea un repositorio nuevo en GitHub.
2. Sube el contenido de esta carpeta a la raiz del repositorio.
3. Verifica que `index.html` quede en la raiz.
4. En GitHub entra a `Settings`.
5. Abre la seccion `Pages`.
6. En `Source` selecciona la rama principal y la carpeta `/root`.
7. Guarda los cambios.
8. Espera unos minutos hasta que GitHub muestre el enlace publico.

## Opcion 2: Netlify por arrastrar y soltar

1. Entra a Netlify.
2. Usa la opcion `Add new site`.
3. Arrastra esta carpeta o el archivo `.zip` generado para publicacion.
4. Netlify publicara el sitio automaticamente.

## Archivo central de contenido

Si quieres editar nombres y descripciones de articulos, hazlo aqui:

- `assets/js/catalog-data.js`

Ese archivo actualiza las tarjetas y la vista ampliada del catalogo.
